## 📖 Overview

<a href="#">Yoink</a> belongs to the category of tools that are invisible in use but immediately missed when absent. The shelf appears exactly when a drag begins and dismisses when it is not needed — integrating with the existing drag workflow rather than requiring a separate step to engage.

The <a href="#">multi-monitor support</a> makes the shelf available on any connected display, enabling cross-display file operations the same way it enables cross-Space operations. <a href="#">Non-destructive holding</a> keeps the original files in place while they are on the shelf — files are not moved or modified until the shelf contents are dropped at a destination, protecting against accidental loss during complex multi-step operations.

<a href="#">Finder tag preservation</a> maintains Finder tags, color labels, and extended attributes on files moved through the shelf, preserving organization metadata that some drag operations strip. <a href="#">Share sheet integration</a> accepts content from macOS's native Share sheet into the Yoink shelf, enabling collection of shared content from applications into the shelf for later placement.

The <a href="#">keyboard shortcut access</a> opens the Yoink shelf from the keyboard for adding clipboard content and URLs without initiating a drag, enabling keyboard-driven use alongside mouse-driven drag operations. <a href="#">iCloud Drive and external volume support</a> handles drag operations involving cloud storage and external drives through the same shelf workflow as local file operations.

## ❓ FAQ

<details>
<summary>Can Yoink be used with keyboard shortcuts rather than drag-and-drop?</summary>

Yes. A keyboard shortcut opens the Yoink shelf for adding clipboard content and URLs without initiating a drag, enabling keyboard-driven use for non-file content.

</details>

<details>
<summary>Does Yoink preserve Finder tags on moved files?</summary>

Yes. Finder tags, color labels, and extended attributes are preserved when files pass through the Yoink shelf to their destination.

</details>

<details>
<summary>Does Yoink work with external drives and iCloud Drive?</summary>

Yes. Files on external volumes and iCloud Drive are handled through the same shelf workflow as local file system operations.

</details>

<details>
<summary>Where does the Yoink shelf appear on screen?</summary>

The shelf position is configurable in Yoink's preferences. It can appear at any screen edge — left, right, top, or bottom — in a position that fits your workflow without interfering with applications.

</details>
